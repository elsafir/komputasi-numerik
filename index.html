soal dan jawaban dalam bentuk code

buatlah program yang dapat mengekspansi bilangan e^2x dengan nilai x=4 hingga nilai menjadi kurang dari 0,001 bisa dibuat dengan listing program sebagai berikut.

import math

x = 4
check = 1
a = 0
b = 1

while check > 0.001 :
    f_x = 0
    f_y = 0
    for i in range(a):
        f_x += (2**i)*x**i/math.factorial(i)

    for j in range(b):
        f_y += (2**j)*x**j/math.factorial(j)

    check = f_y - f_x
    a+=1
    b+=1
    print('iterasi ke-',a,'= ',check)

MATERI 2 NEWTON RAPHSON

Diberi fungsi f (x) pada bilangan mengambang x dan tebakan awal untuk root, temukan root fungsi dalam interval. Di sini f (x) mewakili persamaan aljabar atau transendental.

Untuk kesederhanaan, kami mengasumsikan bahwa turunan fungsi juga disediakan sebagai input.

Kami telah membahas metode di bawah ini untuk menemukan root di set 1 dan set 2 Set 1: Metode Bisection Set 2: Metode Posisi Salah

Perbandingan dengan dua metode di atas:

  1. Dalam metode sebelumnya, kami diberi interval. Di sini kita diminta nilai tebakan awal dari root.
  2. Dua metode sebelumnya dijamin akan menyatu, Newton Rahhson mungkin tidak konvergen dalam beberapa kasus.
  3. Metode Newton Raphson membutuhkan turunan. Beberapa fungsi mungkin sulit mustahil untuk dibedakan.
  4. Untuk banyak masalah, metode Newton Raphson konvergen lebih cepat daripada dua metode di atas.
  5. Juga, ia dapat mengidentifikasi akar berulang, karena tidak mencari perubahan dalam tanda f (x) secara eksplisit

Rumus: Mulai dari tebakan awal x 1 , metode Newton Raphson menggunakan rumus di bawah ini untuk menemukan nilai x berikutnya, yaitu x n +1 dari nilai x x sebelumnya. 

import numpy as np
def f(x):
    return (x/3)**2-np.sin(x)-3
def g(x):
    return (2/9)*x-np.cos (x)
x = np.zeros(20)
x[0]=10

for n in range (19):
    x[n+1]=x[n]-f(x[n])/g(x[n])
    print(x[n],'\t',f(x[n]))
